# Cybersecurity Projects 🔐

A collection of Python security tools built while learning offensive and defensive security concepts.

## 🛠️ Tools

### 1. Dictionary Attack Tool
Cracks passwords by comparing them against a real wordlist.
- Tracks attempts and time taken
- Compatible with real wordlists like rockyou.txt
- Demonstrates why common passwords are dangerously weak

### 2. Port Scanner
Scans a target machine for open ports.
- Identifies which services are running
- Demonstrates the first step of real penetration testing

### 3. Caesar Cipher
Encrypts and decrypts messages using the Caesar cipher method.
- Demonstrates the foundations of cryptography
- Configurable shift value

### 4. Caesar Cipher Brute Forcer
Automatically cracks Caesar encrypted messages without knowing the key.
- Tries all 25 possible shifts
- Demonstrates why Caesar cipher is completely broken as encryption

### 5. Password Brute Forcer
Attempts to crack a password by trying every possible combination.
- Configurable character sets
- Tracks attempts and time taken
- Demonstrates combinatorial explosion

## 💡 Purpose
These tools were built for **educational purposes only** to understand how common attacks work and why strong passwords and security practices matter.

## 🚀 Getting Started
Make sure you have Python installed, then run any file with:
python filename.py
